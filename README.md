 Контрольные вопросы
1️⃣ Как установить PHP?

Скачать PHP с официального сайта, распаковать архив, добавить путь к папке PHP в системную переменную Path и проверить установку через командную строку.

2️⃣ Как проверить, что PHP установлен и работает?

Открыть командную строку и выполнить:

php -v

Если отображается версия PHP — установка выполнена успешно.

3️⃣ Чем отличается echo от print?
echo	print
Может выводить несколько строк	Выводит только одну строку
Работает быстрее	Работает чуть медленнее
Не возвращает значение	Возвращает значение 1
Используется чаще	Используется реже

В большинстве случаев применяют echo.
