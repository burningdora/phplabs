1️⃣ Как установить PHP?

Скачать PHP с официального сайта, распаковать архив, добавить путь к папке PHP в системную переменную Path и проверить установку через командную строку.

Открыть командную строку и выполнить:

php -v

Если отображается версия PHP — установка выполнена успешно.

3️⃣ Чем отличается echo от print?

## Лабораторная работа №5: Объектно-ориентированное программирование
В этой работе я реализовала систему управления банковскими транзакциями.

**Что было сделано:**
* **Интерфейсы:** Создан `TransactionStorageInterface` для гибкости хранения данных.
* **Классы и Объекты:** Написаны классы `Transaction`, `Repository` и `Manager`.
* **Бизнес-логика:** Реализована сортировка по дате, расчет общей суммы и фильтрация.
* **Типизация:** Использован режим `strict_types=1` для защиты от ошибок в типах данных.

### Детальный список выполненных задач в Лабораторной работе №5:

1.  **Архитектура и ООП:**
    * **Создание интерфейса**: Реализован `TransactionStorageInterface`, который определяет обязательные методы для любого хранилища данных (инкапсуляция логики хранения).
    * **Класс-сущность (Entity)**: Разработан класс `Transaction` с приватными свойствами и геттерами для защиты данных.
    * **Слои приложения**: Разделена ответственность между репозиторием (`TransactionRepository`) и менеджером (`TransactionManager`).

2.  **Работа с данными и бизнес-логика:**
    * **Обработка транзакций**: Написана логика добавления объектов транзакций в хранилище через метод `addTransaction`.
    * **Сортировка**: Реализована функция `usort` для автоматической сортировки всех транзакций по дате (от ранних к поздним).
    * **Агрегация**: Добавлен расчет общей суммы всех операций (Total Amount).
    * **Фильтрация**: Реализован подсчет количества транзакций для конкретного получателя (например, KFC).

3.  **Безопасность и стандарты кода:**
    * **Строгая типизация**: Включен режим `declare(strict_types=1)`, что гарантирует соответствие типов передаваемых аргументов и возвращаемых значений.
    * **Type Hinting**: Для всех методов и свойств указаны типы данных (`int`, `float`, `string`, `DateTime`), что минимизирует вероятность ошибок.

4.  **Визуализация (UI):**
    * **Динамическая таблица**: Реализован автоматический вывод массива объектов в HTML-таблицу с использованием циклов.
    * **Форматирование**: Суммы выводятся с фиксированным количеством знаков после запятой, а даты — в удобном для чтения формате.
